# Customer Shopping Behavior Dashboard

## Overview - An interactive analytics dashboard built using Streamlit, PostgreSQL, Pandas, and Plotly to analyze customer shopping behavior.

## Features

* Real-time PostgreSQL data integration
* Interactive filters
* KPI cards
* Customer demographic analysis
* Category-wise sales analysis
* Location-wise sales analysis
* Product popularity analysis
* Purchase amount distribution
* Review rating insights

## Technologies Used

* Python
* PostgreSQL
* Streamlit
* Pandas
* Plotly
* SQLAlchemy

## Dashboard Visualizations

* Gender Distribution
* Age Distribution
* Purchase Amount Distribution
* Sales by Category
* Sales by Location
* Top Purchased Products
* Purchase Amount by Gender
* Age vs Purchase Amount
